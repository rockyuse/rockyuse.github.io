选择框
选择框的宽度自定义 选择框的宽度自定义(默认是200) 选择框的高度自定义(默认是280) 填充数据之前 初始化完毕 初始化选中的值 选择后的回调 反选后的回调 初始化提示文字 可搜索 清除已选项图标 多选 多选的内容在一行 最多选择记录数 超出选择记录的提示 选项连接符 后台搜索

选择框的宽度自定义


					$('#Chosen1').Chosen({
						chosenWidth: 100,
						data: data01
					});

选择框的宽度自定义(默认是200)


					$('#Chosen2').Chosen({
						chosenWidth: 200,
						data: data01,
						dataListWidth: 400
					});

选择框的高度自定义(默认是280)


					$('#Chosen3').Chosen({
						chosenWidth: 200,
						data: data01,
						dataListHeight: 400
					});

填充数据之前


		            $('#Chosen4').Chosen({
						chosenWidth: 200,
						data: data01,
						beforeCallback: function(_self, data){
							console.log(data[0]);
						}
					});

初始化完毕


		            $('#Chosen5').Chosen({
						chosenWidth: 200,
						data: data01,
						initCallback: function(_self, data){
							console.log('填充完毕！');
						}
					});

初始化选中的值


		            $('#Chosen6').Chosen({
						chosenWidth: 200,
						data: data01,
						multi: true,
						initSelectedValue: ['Albania', 'Greenland', 'Iceland'],
					});

选择后的回调


					$('#Chosen7').Chosen({
						chosenWidth: 200,
						data: data01,
						selectedCallback: function(id, name, _self){
							alert('选中了：'+id+'-'+name);
						}
					});

反选后的回调


		            $('#Chosen8').Chosen({
						chosenWidth: 200,
						data: data01,
						removeCallback: function(id, name, _self){
							alert('反选了：'+id+'-'+name);
						}
					});

初始化提示文字


		            $('#Chosen9').Chosen({
						chosenWidth: 200,
						data: data01,
						placeholderTxt: '请选择型号'
					});

可搜索


		            $('#Chosen10').Chosen({
						chosenWidth: 200,
						data: data01,
						searchOpt: true
					});

清除已选项图标


		            $('#Chosen11').Chosen({
						chosenWidth: 200,
						data: data01,
						clearOpt: true
					});

多选


					$('#Chosen12').Chosen({
						chosenWidth: 200,
						data: data01,
						multi: true
					});

多选的内容在一行


		            $('#Chosen13').Chosen({
						chosenWidth: 200,
						data: data01,
						multi: true,
						multiInline: true
					});

最多选择记录数


		            $('#Chosen14').Chosen({
						chosenWidth: 200,
						data: data01,
						multi: true,
						maxSize: 5
					});

超出选择记录的提示


		            $('#Chosen15').Chosen({
						chosenWidth: 200,
						data: data01,
						multi: true,
						maxSizeTip: '只能选择5个型号！'
					});

选项连接符


		            $('#Chosen16').Chosen({
						chosenWidth: 200,
						data: data01,
						multi: true,
						joinChar: '||'
					});

后台搜索


		            //后台搜索
					$('#Chosen17').Chosen({
						chosenWidth: 200,
						data: data01,
						searchOpt: true,
						searchBack: true,
						searchAjaxURL: 'getInfo.do',
						ajaxParam: {'name': 'Tom'}
					});
